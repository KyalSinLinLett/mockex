    Mockex (mockex v0.1.0)

    Mockex is a lightweight mocking library for elixir. Create simple mocks for external/third-party calls or mock websocket servers with custom listeners and responses.

    use_mock(mock_module, opts \\ [], mocks, list)

    (macro)

    Macro used to define a mock. Provide the Module, function to be mocked, and the expectation. Then, make function calls as you normally would.

    examples

     Examples 

    test "mock the String.at/2 function" do
  use_mock String,
    at: fn _, _ -> "mock response here" end do
    assert String.at("hello", 2) == "mock response here"
  end
end
    Link to this macro

    use_mocks(mocks, list)

    (macro)

    Macro used to define several mock modules before each test. Provide the Module, function to be mocked, and the expectation. Then, make function calls as you normally would.

    examples

     Examples 

    test "mock testing several modules and their functions" do
  use_mocks([
    {Map, [],
    [
      get: fn %{}, "key" -> "value" end
    ]},
    {String, [],
    [
      reverse: fn _ -> :reversed end,
      length: fn _ -> :some_length end
    ]}
  ]) do
    assert Map.get(%{}, "key") == "value"
    assert String.reverse(3) == :reversed
    assert String.length(3) == :some_length
  end
end
